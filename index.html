DNN xChange - Moved from Codeplex

DNN xChange is a Q&A module and more! The new release introduces tight integration with DNN Social Groups, improves profile integration. We are currently working on improving and modernizing UI. We are looking for Web designers (HTML5/CSS3/jQuery) to improve on the DNN xChange UI.

Key features:

  • Profile integration
  • Journal integration
  • Social Groups integration
  • Tags based on Core Taxonomy
  • Tag Synonyms
  • Tag Suggest when adding question
  • Question Suggest: Questions with similar titles
  • Subscriptions to Questions by Tag
  • Answer Notifications
  • Comments
  • Voting up and down
  • Social sharing (Google+, Tweeter, LinkedIn, Facebook)
  • Flagging inappropriate questions/answers
  • Configurable Scoring
  • Customisable Privilege assignment
  • Top Questions, Unanswered Questions
  • My Dashboard
  • Recent Tags
  • Badges.......coming soon!

Developers and Designers are needed!
